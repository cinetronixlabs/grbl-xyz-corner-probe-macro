# GRBL XYZ コーナープローブマクロ

これは、導通式のXYZタッチプローブブロックを使って、ワーク座標の原点を設定するためのGRBL互換XYZコーナープローブマクロです。

このマクロでは、最初にZを測定し、その後、選択したコーナー方向に応じてXとYを測定します。  
材料天面を `Z0` に設定し、ワークのコーナー位置が `X0 Y0` になるように現在のワーク座標系を設定します。

プロービングには、`G38.2` や `G10 L20` などのGRBL互換G-codeコマンドを使用しています。

このマクロは、特定の制御ソフト専用ではありません。  
gSender、CNCjs、その他のGRBL制御ソフトでも使用できる可能性がありますが、変数や計算式の書式は各ソフトのマクロ機能に依存します。

## マクロファイル

実際のマクロはこちらです。

[`xyz-corner-probe-ja.nc`](./xyz-corner-probe-ja.nc)

## このマクロでできること

- XYZプローブブロックのコーナー穴から開始
- プローブブロック上面でZを測定
- 材料天面を `Z0` に設定
- 選択したX側を測定
- 工具半径とプローブブロック厚みを考慮してX座標を設定
- 選択したY側を測定
- 工具半径とプローブブロック厚みを考慮してY座標を設定
- 設定された `X0 Y0` 位置へ移動

## テスト環境

このマクロは、GRBLベースのデスクトップCNC環境で作成・テストしました。

使用例:

- 機械: LUNYEE 3020 Nova
- コントローラー: GRBLベースのコントローラー
- ファームウェア: GRBL互換 1.1f
- プローブ: 導通式XYZタッチプローブブロック
- 工具: 導通するエンドミル
- プローブ入力: 工具がプローブブロックに接触したときに反応

この環境では、導通式のXYZタッチプローブブロックを使用しています。

使用したプローブブロックの例:

- XYZタッチプローブブロック: https://s.click.aliexpress.com/e/_c4OqYFct

このリンクはアフィリエイトリンクです。アクセス元の国に応じたAliExpressの最適な地域ドメインへ自動リダイレクトするために使用しています。

## LUNYEE 3020 Novaでの接続例

これは、導通式XYZタッチプローブブロックの基本的な接続例です。

このタイプのプローブブロックは、工具とプローブブロックが接触して導通したときにプローブ入力が反応します。

```text
XYZタッチプローブブロック        3020 Nova

プローブブロック本体 ----------> PROBE S
工具 / エンドミルのクリップ ----> PROBE -
```

この接続では、

- プローブブロック本体を `PROBE S` に接続
- 工具またはエンドミルをワニ口クリップなどで `PROBE -` に接続
- 工具がプローブブロックに触れると、`PROBE S` が `PROBE -` に導通し、プローブ入力が反応します

環境によっては、本体シャーシ上の導通しているGND部分に `PROBE -` を接続しても動作する場合があります。  
ただし、その場合は工具まで確実に導通している必要があります。

安定性を重視する場合は、工具またはエンドミルに直接ワニ口クリップで接続する方法を推奨します。

マクロを実行する前に、制御ソフト上でプローブ入力をテストし、工具がプローブブロックに触れたときに状態が変化することを確認してください。

## 互換性について

このマクロで使用しているプロービングコマンド自体は、GRBL互換の標準的なG-codeコマンドです。

ただし、以下のような変数や計算式の書式は、制御ソフト側のマクロ機能に依存します。

```gcode
%endmillDiameter = 6.35
G0 Z[safeZ]
G38.2 X[xyProbeTravel * sx] F[fast]
```

使用している制御ソフトがこの書式に対応していない場合は、そのソフトのマクロ形式に合わせて書き換えるか、変数部分を固定の数値に置き換える必要があります。

## コーナー方向設定

このマクロは、`sx` と `sy` の値を変更するだけで、異なるワークコーナーに対応できます。

初期設定:

```gcode
%sx = 1
%sy = -1
```

この初期設定は、左奥コーナー用です。

| コーナー | `sx` | `sy` |
|---|---:|---:|
| 左手前 | `1` | `1` |
| 左奥 | `1` | `-1` |
| 右手前 | `-1` | `1` |
| 右奥 | `-1` | `-1` |

このマクロでは、

- `sx` がX方向のプローブ方向とX座標の符号を制御します
- `sy` がY方向のプローブ方向とY座標の符号を制御します

これらの値は、機械の実際の X+ / Y+ 方向を基準にしています。  
原点復帰位置そのものには依存しません。

マクロを実行する前に、手動ジョグで X+ と Y+ が実際にどちらへ動くかを確認してください。  
機械の軸方向が異なる場合は、実行前に必ずプロービング方向を確認し、必要に応じて設定を調整してください。

## 座標方向の概要

```text
上面図

                  Y+
                  ^
                  |
                  |

        +-------------------+
        |                   |
        |       材料        |
        |                   |
        +-------------------+

X- <---------------------------------> X+
```

初期設定の左奥コーナーの場合:

```gcode
%sx = 1
%sy = -1
```

この場合、

- X側はX+方向へプローブします
- Y側はY-方向へプローブします

## 実行前の開始位置

![XYZコーナープローブのセットアップ例](./assets/xyz-corner-probe-setup.png)

マクロを実行する前に、以下を確認してください。

1. エンドミルをプローブブロックのコーナー穴の中心に入れる
2. 工具先端がプローブブロック上面より下にある状態にする
3. スピンドルがOFFであることを確認する
4. プローブ入力が正しく動作していることを確認する
5. 測定するコーナーに対して `sx` と `sy` の設定が正しいことを確認する

このマクロは、工具がプローブブロックのコーナー穴の中から開始することを前提にしています。

## パラメータ

`xyz-corner-probe-ja.nc` 内の以下の値を、使用する工具、プローブブロック、環境に合わせて変更してください。

```gcode
%endmillDiameter = 6.35
%toolR = endmillDiameter / 2

%probeBlockZ = 5.04
%probeBlockX = 10.03
%probeBlockY = 10.00

%fast = 70.0
%slow = 30.0

%sx = 1
%sy = -1

%safeZ = 13.0

%zProbeTravel = 30.0
%zBackoff = 2.0
%zReprobeTravel = 5.0
%zLiftAfterProbe = 3.0

%zTouchDistanceX = 10.0
%zTouchDistanceY = 10.0

%xyApproach = 20.0
%xyProbeTravel = 45.0
%xyDepthBelowPlateTop = 3.0
%xyBackoff = 2.0
%xyReprobeTravel = 8.0
%xyOutClear = 10.0
%xyInsetFromEdge = 5.0
```

## パラメータ説明

| パラメータ | 説明 |
|---|---|
| `endmillDiameter` | プロービングに使用するエンドミル径 |
| `toolR` | `endmillDiameter` から計算される工具半径 |
| `probeBlockZ` | Z測定に使用するプローブブロックの高さ、または厚み |
| `probeBlockX` | プローブブロックのX方向の厚み |
| `probeBlockY` | プローブブロックのY方向の厚み |
| `fast` | 1回目のプロービング送り速度 |
| `slow` | 2回目の低速プロービング送り速度 |
| `sx` | 選択したコーナーに対するX方向設定 |
| `sy` | 選択したコーナーに対するY方向設定 |
| `safeZ` | コーナー穴から工具を抜くためのZ退避量 |
| `zProbeTravel` | Zプローブ時の最大下降量 |
| `zBackoff` | 1回目のZプローブ後の戻り量 |
| `zReprobeTravel` | 2回目のZプローブ時の最大下降量 |
| `zLiftAfterProbe` | Z設定後に上げる量 |
| `zTouchDistanceX` | コーナー穴からZ測定位置までのX方向距離 |
| `zTouchDistanceY` | コーナー穴からZ測定位置までのY方向距離 |
| `xyApproach` | X/Y測定側の外側へ回り込むための追加距離 |
| `xyProbeTravel` | X/Yプローブ時の最大探索距離 |
| `xyDepthBelowPlateTop` | X/Y測定時にプローブブロック上面から下げる量 |
| `xyBackoff` | 1回目のX/Yプローブ後の戻り量 |
| `xyReprobeTravel` | 2回目のX/Yプローブ時の最大探索距離 |
| `xyOutClear` | 接触面から逃がす距離 |
| `xyInsetFromEdge` | Y測定前に、測定済みのX端面から内側へ入る距離 |

## Zプローブの動作

このマクロでは、最初に工具をコーナー穴から上へ抜きます。

```gcode
G0 Z[safeZ]
```

その後、Z測定位置へ移動します。

```gcode
G0 X[zTouchDistanceX * sx] Y[zTouchDistanceY * sy]
```

Zを2回測定します。

```gcode
G38.2 Z[-zProbeTravel] F[fast]
G0 Z[zBackoff]
G38.2 Z[-zReprobeTravel] F[slow]
```

その後、現在のZ位置をプローブブロックの高さに設定します。

```gcode
G10 L20 P0 Z[probeBlockZ]
```

これにより、材料天面が `Z0` になります。

## Xプローブの動作

Z測定後、マクロは選択したX側の外側へ移動します。

```gcode
G0 X[-(zTouchDistanceX + xyApproach) * sx]
```

その後、X/Y測定用の高さまで下げます。

```gcode
G0 Z[-(zLiftAfterProbe + xyDepthBelowPlateTop)]
```

Xを2回測定します。

```gcode
G38.2 X[xyProbeTravel * sx] F[fast]
G0 X[-xyBackoff * sx]
G38.2 X[xyReprobeTravel * sx] F[slow]
```

その後、X座標を設定します。

```gcode
G10 L20 P0 X[-(toolR + probeBlockX) * sx]
```

この設定では、以下を補正しています。

- 工具半径
- プローブブロックのX方向厚み
- 選択したX側の方向

## Yプローブの動作

Yを測定する前に、マクロは選択したY側の外側へ移動します。

```gcode
G0 Y[-(zTouchDistanceY + xyApproach) * sy]
```

その後、測定済みのX端面から少し内側へ移動します。

```gcode
G90
G0 X[xyInsetFromEdge * sx]
G91
```

Yを2回測定します。

```gcode
G38.2 Y[xyProbeTravel * sy] F[fast]
G0 Y[-xyBackoff * sy]
G38.2 Y[xyReprobeTravel * sy] F[slow]
```

その後、Y座標を設定します。

```gcode
G10 L20 P0 Y[-(toolR + probeBlockY) * sy]
```

この設定では、以下を補正しています。

- 工具半径
- プローブブロックのY方向厚み
- 選択したY側の方向

## ワーク座標の設定

このマクロでは、以下のコマンドを使用しています。

```gcode
G10 L20 P0
```

これは、現在使用中のワーク座標系を設定します。

G54を明示的に設定したい場合は、

```gcode
G10 L20 P0
```

を、

```gcode
G10 L20 P1
```

に変更できます。

例:

```gcode
G10 L20 P1 Z[probeBlockZ]
G10 L20 P1 X[-(toolR + probeBlockX) * sx]
G10 L20 P1 Y[-(toolR + probeBlockY) * sy]
```

## 安全上の注意

このマクロは自己責任で使用してください。

実際の材料上で実行する前に、以下を必ず確認してください。

- プローブ入力を事前にテストする
- プローブ配線を確認する
- 機械の実際の X+ / Y+ 方向を確認する
- プロービング方向を確認する
- 測定するコーナーに対して `sx` と `sy` の設定が正しいことを確認する
- 工具がプローブブロックのコーナー穴の中から開始していることを確認する
- スピンドルがOFFであることを確認する
- 工具が導通し、プローブ入力を反応させられることを確認する
- プローブブロックの寸法が正しいことを確認する
- 初回テスト時は慎重に実行する
- いつでも停止できる状態で実行する

プローブブロック寸法の誤り、コーナー方向設定の誤り、軸方向の認識違い、開始位置の誤りは、機械の衝突や誤ったワーク座標設定につながる可能性があります。

このマクロはGRBLベースのデスクトップCNC環境で作成・テストしましたが、動作はコントローラー、ファームウェア、制御ソフト、マクロパーサー、プローブ入力の設定によって変わる可能性があります。
